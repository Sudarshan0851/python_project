#User Authentication System (Python)

This is a basic command-line user authentication system I built using Python.
It lets you register, login, and reset your password.

---

## What it does

* Register new users
* Login with email + password
* Reset password if forgotten
* Stores data in a file (`users.txt`)
* Passwords are hashed (not stored as plain text)

---

## How it works

When you run the program, you’ll see options like:

* Register
* Login
* Forgot Password
* View users (for testing)
* Exit

You just enter the number and follow the steps.

---

## Rules

* Email must be valid
* Password must:

  * be at least 6 characters
  * have 1 uppercase letter
  * have 1 special character

---

## Run it

```bash
python main.py
```
